# 🤖 Introdução ao Ansible

![Ansible](https://www.ansible.com/hubfs/ansible-logo.png)

O **Ansible** é uma ferramenta de **automação de TI** que permite gerenciar configuração, deploy de aplicações e orquestração de infraestrutura de forma simples e sem agentes. Ele usa **YAML** para descrever tarefas e playbooks.

---

## 🚀 Vantagens do Ansible

- Automação sem agentes  
- Fácil de aprender (YAML simples)  
- Escalável para grandes infraestruturas  
- Integração com cloud, containers e redes  
- Reutilização de playbooks e roles

---

## 💡 Comandos básicos

| Comando               | Função                                |
|----------------------|---------------------------------------|
| `ansible -m ping all` | Testa conectividade com hosts          |
| `ansible-playbook site.yml` | Executa playbooks                 |
| `ansible-galaxy init myrole` | Cria uma role                        |
| `ansible-doc <module>` | Mostra documentação de módulos        |
